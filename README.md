# Satellite Image Damage Detection System

An automated system to identify structural damage in satellites and track orbital debris using YOLOv8.

## Project Roadmap
- [x] Phase 1: Environment Setup
- [ ] Phase 2: Data Collection & Labeling
- [ ] Phase 3: Model Training
- [ ] Phase 4: Tracking & Collision Logic
- [ ] Phase 5: Dashboard Deployment

## Setup Instructions
1. Clone the repo
2. Create venv: `python -m venv venv`
3. Activate: `.\venv\Scripts\activate`
4. Install: `pip install ultralytics opencv-python pandas matplotlib`
